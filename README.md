# painel-personagem
Painel De Personagens

Este projeto consiste em um painel interativo para votação de personagens. Nele, os usuários podem visualizar a lista de personagens disponíveis, selecionar seus favoritos e acompanhar o resultado das votações em tempo real. O objetivo é fornecer uma interface simples, clara e funcional para enquetes, competições ou análises de preferência do público.

# Como Rodar

        1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
        2. Extraia a pasta
        3. Abra o arquivo index.html no navegador

# Como Clonar

No terminal do VS Code digite:
	git clone https://github.com/Fellipe-07/painel-personagem.git
No GitHub Desktop:
	No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/Fellipe-07/painel-personagem.git
